# 🤖 RL Biped

A reinforcement learning project aimed at teaching a bipedal robot model to walk using a reward-based training approach. The agent is penalized for falling and rewarded for successfully maintaining balance and moving forward.

---

## 📋 Overview

This project uses a custom MuJoCo environment to simulate a bipedal robot. Through thousands of iterations of trial and error, the agent learns to walk by receiving rewards for forward movement and penalties for falling. The biped model is defined using MuJoCo's XML format and trained using Python.
